# TreeQueryER

## Run FrontEnd
```
cd ./FrontEnd
```

#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production (only for finishing the whole project)
```
npm run build
```

## Run BackEnd
```
cd ./BackEnd
```

#### Project setup
```
python communicator.py
```
